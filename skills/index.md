# skills/index.md
# 스킬 목록 인덱스

모든 페르소나는 이 파일을 가장 먼저 읽어야 한다.
필요한 스킬 파일을 이 목록에서 찾아 해당 파일을 읽고 참고한다.

---

## 페르소나별 필수 스킬

| 페르소나 | 필수 스킬 파일 | 용도 |
|---------|--------------|------|
| 계획자 | `skills/planner.md` | 요구사항 분석, 계획서 작성 방법 |
| 생성자 | `skills/generator.md` | 코드 작성 방법, 구조 설계 |
| 평가자 | `skills/evaluator.md` | 오류 검출, 보안 검토, 품질 기준 |

---

## 스킬 파일 설명

### planner.md
- 사용자 요구사항 분석 방법
- 계획서 작성 템플릿
- 사용자와 피드백 주고받는 방법
- 기획서.txt 저장 형식

### generator.md
- 코드 작성 전 준비사항 체크리스트
- 파일 구조 설계 방법
- 작업일지.txt 작성 형식
- 오류보고서.txt 참고해서 재작업하는 방법

### evaluator.md
- CMD 실행 테스트 방법
- 오류 유형별 체크리스트
- 보안 검토 항목
- 오류보고서.txt 작성 형식
- 재시도 횟수 관리 방법
